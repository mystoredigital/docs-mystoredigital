Acceso a admin.ecuabet.com mediante proxy con IP fija

Guía paso a paso para que los operadores y la agencia se conecten al panel de Ecuabet a través del proxy con IP autorizada de My Store Digital.

Fecha · 2026-06-12 Sistema · Proxy HTTP/HTTPS · Firefox · FoxyProxy
Resumen

Qué es: un proxy con una IP fija autorizada que te deja entrar a admin.ecuabet.com aunque tu internet de casa cambie de IP.

Por qué: Ecuabet bloqueó el panel por seguridad y solo permite el acceso desde IPs autorizadas (whitelist). Tu IP de casa cambia sola, así que no se puede autorizar; el proxy sí tiene una IP fija.

Qué haces: configuras el proxy en tu navegador con tu usuario y contraseña, verificas tu IP, operas, y al terminar lo desactivas.

Antes de empezar

Necesitas tus credenciales personales del proxy. Cada operador tiene las suyas — pídelas al administrador si todavía no las tienes:

Servidor (Host)
150.230.28.174
Puerto
8443
Usuario
TU_USUARIO (te lo da el administrador)
Contraseña
TU_CONTRASEÑA (te la da el administrador)
Info. El proxy funciona en cualquier red (casa, oficina, datos móviles). Lo único que importa es que pongas bien el usuario y la contraseña.

¿Por qué pasa esto?

El panel admin.ecuabet.com sufrió un ataque y, como medida de seguridad, ahora solo acepta conexiones que vengan desde una IP autorizada. El problema: los servicios de internet domésticos entregan IPs dinámicas (cambian solas cada cierto tiempo), por lo que no se puede autorizar tu IP de casa.

La solución es pasar tu conexión por un proxy de My Store Digital que tiene una IP fija (150.230.28.174). Esa IP sí está autorizada por Ecuabet. Cuando activas el proxy, Ecuabet ve esa IP fija en lugar de la tuya y te deja entrar.

Opción A — Configurar en Firefox (recomendado)

Firefox es la opción más limpia porque el proxy solo afecta a ese navegador, no al resto de tu computadora.

  1. Abre Firefox y entra a Ajustes (menú ☰ arriba a la derecha → Ajustes).
  2. En el buscador de ajustes escribe proxy y haz clic en Configuración… (sección "Configuración de red").
  3. Marca la opción Configuración manual del proxy.
  4. En Proxy HTTP escribe 150.230.28.174 y en Puerto escribe 8443.
  5. Marca la casilla "Usar también este proxy para HTTPS".
  6. Haz clic en Aceptar para guardar.
  7. Entra a admin.ecuabet.com. Firefox te pedirá usuario y contraseña → escribe los tuyos (TU_USUARIO / TU_CONTRASEÑA).
Advertencia. Si Firefox no te pide usuario y contraseña y la página no carga, revisa que escribiste bien el host y el puerto. La contraseña distingue mayúsculas y minúsculas.

Opción B — Configurar con FoxyProxy

FoxyProxy es una extensión que permite activar el proxy solo para el panel de Ecuabet y dejar el resto de tu navegación normal. Útil si usas el mismo navegador para todo.

  1. Instala la extensión FoxyProxy Standard (disponible para Chrome y Firefox).
  2. Abre FoxyProxy → Options → Add (agregar proxy nuevo).
  3. Completa los datos:
    • Title: Ecuabet
    • Proxy Type: HTTP
    • Hostname / IP: 150.230.28.174
    • Port: 8443
    • Username: TU_USUARIO
    • Password: TU_CONTRASEÑA
  4. Guarda. Luego agrega un patrón de URL para que el proxy se active solo en Ecuabet:
    • En el proxy creado → Patterns → Add
    • Pattern: *admin.ecuabet.com* · Type: Wildcard
  5. En el ícono de FoxyProxy elige el modo "Use proxies based on their patterns". Así solo Ecuabet pasa por el proxy.

Verificar que funciona

Antes de operar, confirma que tu conexión está saliendo por la IP autorizada:

  1. Con el proxy activo, entra a ipinfo.io.
  2. Verifica que la IP que muestra sea 150.230.28.174.
OK. Si ves 150.230.28.174, el proxy está funcionando y ya puedes entrar a admin.ecuabet.com con normalidad.
Riesgo. Si ves otra IP, el proxy NO está activo y Ecuabet te seguirá bloqueando. Revisa la configuración antes de continuar — no operes hasta ver la IP correcta.

Checklist antes de operar

  1. Tengo mi usuario y contraseña del administrador.
  2. Configuré el proxy 150.230.28.174 : 8443 en Firefox o FoxyProxy.
  3. Marqué "usar también para HTTPS" (Firefox) o el patrón de Ecuabet (FoxyProxy).
  4. Verifiqué en ipinfo.io que mi IP es 150.230.28.174.
  5. El panel admin.ecuabet.com carga correctamente.

Al terminar de operar

Importante. Cuando termines de trabajar en Ecuabet, desactiva el proxy para que tu navegación normal no pase por él:
  • Firefox: Ajustes → Configuración de red → marca "Sin proxy" → Aceptar.
  • FoxyProxy: haz clic en el ícono y elige "Turn off" (desactivar).

Problemas comunes

No me pide usuario/contraseña y la página no carga

Revisa que el host sea exactamente 150.230.28.174 y el puerto 8443. Un espacio de más o un número mal escrito impide la conexión.

Me pide usuario/contraseña pero las rechaza

La contraseña distingue mayúsculas, minúsculas y números. Cópiala y pégala con cuidado. Si sigue fallando, pide al administrador que verifique o regenere tu clave.

ipinfo.io muestra otra IP, no la 150.230.28.174

El proxy no está activo. En Firefox confirma que está marcada la "Configuración manual del proxy"; en FoxyProxy confirma que el modo por patrones está activo y que estás dentro de una URL de Ecuabet.

Mi navegación normal (otras webs) se volvió lenta

Probablemente dejaste el proxy activo para todo. Usa FoxyProxy con el patrón de Ecuabet, o desactiva el proxy de Firefox cuando no estés en el panel.

Soporte. Si después de revisar todo sigues sin poder entrar, contacta al administrador de My Store Digital con una captura de lo que muestra ipinfo.io.
